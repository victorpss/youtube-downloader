# youtube-downloader
download mp3, mp4 and gif


## instalações

> pip install yt-dlp moviepy==1.0.3

### baixar também o [ffmpeg](https://www.ffmpeg.org/download.html)

> acessar Windows builds from gyan.dev
> 
> baixar ffmpeg-7.0.2-essentials_build.zip
> 
> extrair em alguma pasta
> 
> copiar o caminho completo da pasta /bin
> 
> acessar **Variáveis de Ambiente** -> **Variáveis do Sistema**
> 
> Path -> Editar -> Novo e colar o caminho
> 
> Dar OK em tudo e verificar no cmd: `ffmpeg -version`

## código

> adicionar as URLs na var 'urls', separando por vírgulas
>
> mudar o formato para "mp3", "mp4" ou "gif"
